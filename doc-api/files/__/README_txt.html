Contributing

To create Gemfile.lock file, run "bundle install". To run the tests, type "rake". To run the tests with (simplecov) coverage, type "COVERAGE='yes' rake". Then open **coverage/index.html** in a browser.

Installation

Add this line to your application's Gemfile: * gem "rkwalify" Or install direcly with "gem install rkwalify".
